data we collect (expanded)

IP addresses are collected automatically to estabilsh a connection to the Notely cloud service / the website. This information is only kept for as long as necssasary (the duration you are actively using the site).

Emails and passwords are collected to allow users to access the cloud service we offer, this data is retained up until the point the user requests to delete their account. Passwords are encrypted and are stored securely at all times.

The data inside notes is collected so users can save their notes to our cloud serivce allowing them to access them on any device with an internet connection. Notes are retained until a user deletes them.

data security practices

When connected to the Notely cloud service any data you send will be securely encrypted in transit.
